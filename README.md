# Car Price Prediction ML Project

This project uses machine learning to predict car prices based on various features.

Features-
- Predicts car prices using ML model
- Data preprocessing and training included
- Simple and beginner-friendly implementation

Technologies Used-
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

How to Run-
1. Clone the repository
2. Install required libraries
3. Run the Python file
